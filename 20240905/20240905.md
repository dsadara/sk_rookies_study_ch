### 파이썬 소개

귀도 반 로섬이 크리스마스때 개발

파이썬이 널리 사용되는 이유
* 웹개발, 데이터 과학, 인공 지능, 과학 계산에 쓰임
    * python으로 만든 Flask, Django 웹 프레임워크
    * Pandas 데이터분석, Tensorflow 머신러닝

* 직관적인 문법

* 교육용 언어로의 인기
    * 대학교에서 입문용 언어로 파이썬 선택
    * PYPL 인기언어 지표 -> 파이썬 1위

### 파이썬 주요 특징

1. 인터프리터 언어   
컴파일 필요 없음

2. 동적 타이핑   
변수형을 사전에 선언할 필요 없음 -> 변수에 문자열, 숫자 아무거나 넣어도 됨
    
3. 객체지향 프로그래밍 지원   
클래스 상속 인스턴스등 개체지향 주요 개념 지원

3. 확장성   
C/C++ 코드를 파이썬 내 사용가능

4. 플랫폼 독립적   
다양한 운영체제에서 동일하게 실행 가능 -> 컴파일 언어는 운영체제마다 컴파일러가 다름

### 파이썬 REPL(Read-Eval_Print Loop)
사용자로부터 코드를 읽고(read), 평가(Evaluate) 하고 출력(Print)한 후, 코드 입력을 기다리는(Loop) 과정을 반복하는 인터프리터
* 컴파일 안하고 바로바로 코드 실행 가능   
* .py를 사용하는 방식은 REPL이 아님

### 파이썬 버전 별 차이점
요즘은 3버전대 많이 사용

* 파이썬 2.x   
ASCII 코드만 지원 -> 즉, 128개 문자만 지원
* print 문법에 괄호가 없음   
print "Hello World"

* 파이썬 3.x   
유니코드 기반으로 더 많은 문자 지원

### IT 보안에서 파이썬 사용하는 사례

1. 네트워크 보안, 웹 보안, 시스템 모니터링 및 로깅, 취약점 분석
    * 공격 프로그램 파이썬으로 작성 (익스플로잇 코드)

    * 네트워크 트래픽 모니터링

    * SQLMAP: SQL 인젝션 취약점 스캔하는 프로그램

4. 포렌식 및 사고 대응   
    * 파이썬 스크립트를 사용하여 로그 파일 이상행위 추출 (정규식 사용)

5. 네트워크 보안   
    * Nmap 네트워크 분석 도구 -> 파이썬으로 개발함

6. 정기적 포트 스캔 결과통보
    * 이메일, 슬랙으로 결과 자동 전송

7. 보안 이슈 자동화 크롤링하여 보고   
    * 엑셀 데이터 보고서로 자동 변환해서 이메일, 슬랙, 카톡으로 전송

### 문자열 인덱싱과 슬라이싱

문자열의 특정 부분에 접근하거나 부분적으로 추출 가능

1. 문자열 인덱싱   
0으로 인덱스 시작
    * -1로 마지막 문자 접근 가능
    
2. 문자열 슬라이싱   
[시작 인덱스:끝 인덱스:스텝]

### 문자열 메서드
1. 대소문자 변환   
* upper()   
* lower()
* 확장자 우회 공격기법 막을 수 있음
    * .jsp .JSP .jSP -> 전부 소문자로 바꾸면 방어 가능
        
2. 문자열 나누기 split()   
separator를 기준으로 나누고 리스트 반환
    * separator 기본은 공백임
        
3. 문자열 시작과 끝 확인
    * startswith(): ~로 시작하는지 확인
        * 예시: 코드에 주석 탐지 사용 가능 startswith("#")

    * endswith(): ~로 끝나는지 확인
        * 확장자 검증시 사용 .jsp .JSp

4. 공백 제거
* lstrip(): 왼쪽 공백 제거
* rstrip(): 오른 공백 제거
* strip(): 양쪽 공백 제거
